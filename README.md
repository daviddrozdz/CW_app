WORKING with the app (from the video):
1. start project (in the terminal)
- npx nuxi@latest init app

2. naviagte to the "app" folder
- cd app

3. install dependencies
- npm install
- npm install @nuxtjs/tailwindcss
- npm install nuxt-icon

4. start to run 
- npm run dev
